## Qiara Code Repository

This repository is used to ONLY provide overview of deployed Smart Contracts on EVM chains and Packages with certain Modules on Move chains.

### Setup Instructions
1. Clone the repo: `git clone https://github.com/cnhuya/QIARA-CODE`

*All Solidity based contracts for EVM blockchain are deployed via Remix Online IDE (https://remix.ethereum.org/)*
